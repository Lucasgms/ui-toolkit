# Grid System

## How it works
Our grid system is just an extension of Bootstrap's grid system.
It also uses containers, rows and columns to layout and align responsiveness content.
You can consult [Bootstrap's grid system documentation](https://getbootstrap.com/docs/4.5/layout/grid/) for further details.

## Options

|                     | Extra small(< 576px )   | Small(>=576px)       | Medium(>= 768px)      | Large(>= 992px)       | Extra large(>=1280px) |
| ------------------- | ----------------------- | -------------------- | --------------------- | --------------------- | --------------------- |
| Max container width | None(auto)              | 540px                | 720px                 | 960px                 | 1140px                |   
| Class prefix        | `.c-col`                | `.c-col-sm`          | `.c-col-md`           | `.c-col-lg`           | `.c-col-xl`           |
| # of columns        | 12                      | 12                   | 12                    | 12                    | 12                    | 
| Gutter width        | 30px                    | 30px                 | 30px                  | 30px                  | 30px                  |
| Nestable            | Yes                     | Yes                  | Yes                   | Yes                   | Yes                   | 
| Column ordering     | Yes                     | Yes                  | Yes                   | Yes                   | Yes                   |
