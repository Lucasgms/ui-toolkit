# Miscellaneous

## border-radius

Default `border-radius` used.

| value | Variable               |
| ----- | ---------------------- |
| 4px   | `var(--border-radius)` |

## transition-duration

The more common `transition-duration` used.

| value | Variable                            |
| ----- | ----------------------------------- |
| 0.25s | `var(--transition-duration: 0.25s)` |

## z-index

All `z-index` variables to maintain consistency.

| value | Variable                |
| ----- | ----------------------- |
| 1     | `var(--zindex-1)`       |
| 10    | `var(--zindex-10)`      |
| 100   | `var(--zindex-100)`     |
| 1000  | `var(--zindex-1000)`    |
| 10000 | `var(--zindex-overlay)` |
