# Typography

The typography used and their CSS Variables.

<Typeset
  fontSizes={['14px', '16px', '24px', '36px', '48px']}
  sampleText="Design System Contabilizei"
/>

## Using

### Variable values

| Variable             | Value                         |
| -------------------- | ----------------------------- |
| `var(--font-family)` | 'Lato', 'Roboto', sans-serif' |

| Variable                         | Font size | Line height | Font weight |
|----------------------------------|-----------|-------------|-------------|
| var(--typography-panel-font)     | 48px      | 58px        | 700         |
| var(--typography-title-font)     | 36px      | 43px        | 700         |
| var(--typography-subheader-font) | 24px      | 29px        | 700         |
| var(--typography-body-font)      | 16px      | 19px        | 400         |
| var(--typography-body2-font)     | 16px      | 28px        | 400         |
| var(--typography-caption-font)   | 14px      | 28px        | 400         |
| var(--typography-button-font)    | 16px      | 19px        | 700         |

### Class values

| Typography           | Value               |
| -------------------- | ------------------- |
| Panel                | `c-t-panel`         |
| Title                | `c-t-title`         |
| Subheader            | `c-t-subheader`     |
| Body                 | `c-t-body`          |
| Body Bold            | `c-t-body-bold`     |
| Body2                | `c-t-body2`         |
| Button               | `c-t-button`        |

We have variables to font and letter spacing, you just need to use on `font` and `letter-spacing` property. You can just use a class to add a typography styles too.

<Preview style={{
  padding: `var(--space-xs) var(--space-sm)`
}}>
  <div style={{
    font: `var(--typography-panel-font)`,
    letterSpacing: `var(--typography-panel-letter)`
  }}>Panel</div>
</Preview>

```html
<h1 class="c-t-panel">Panel</h1>
```

```css
foo {
  font: var(--typography-panel-font);
  letter-spacing: var(--typography-panel-letter);
}
```
