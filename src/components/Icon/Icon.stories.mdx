# Icon

Icons can be used in many contexts. Icon components accept size and variants props.

## Examples
Default icons have:
* Size: default application space variable
* Color: default application text color

<Canvas>
  <Story name="Default" parameters={{ decorators: [withA11y] }} inline={true}>

          <Icon icon="done"></Icon>

        </div>

    }}
  </Story>
</Canvas>


### Variants
To change the icon's color, variant prop can be:
* Empty: default text color
* Primary: default primary color
* White: default white color

<Canvas>
